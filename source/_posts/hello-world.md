欢迎来到我的新博客。

这个站点使用 [Hexo](https://hexo.io/) 生成，主题是 [Butterfly](https://butterfly.js.org/)，发布目标是 GitHub Pages。接下来可以把它变成技术笔记、项目日志、阅读摘录，或者任何值得长期保存的东西。

## 常用命令

本地预览：

```bash
npm run server
```

生成静态文件：

```bash
npm run build
```

新建文章：

```bash
npx hexo new "文章标题"
```
